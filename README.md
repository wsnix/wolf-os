wolf os , es un mini sistema operativo escrito en c++.
creado para correr en la terminal de linux.

en esta primera version el sistema es un simple archivo bin compilado con otros archivos/bins , y algunos de terminacion, txt etc.

la idea central es crear un os exclusivamente para terminal con uan interfaz de ascii, las funciones que pienso añadir son.

1. conexion wifi funcional y real.
2. interfaz sencilla hecha en ascii
3. navegacion real en internet , aunque no espere algo tan complejo quiza solo de lectura.
4. y la posibilidad de instalar otros programas ademas de lo que cree para el os. pero no estoy seguro de como hacer eso o si quiera se podra hacer.
